<h1>🎓 양동이 챗봇 (Yangdongyi Chatbot)</h1>
<p>동양미래대학교 학사 정보를 더 쉽고, 더 빠르게, 더 정확하게 전달하기 위한 대학생 전용 AI 도우미</p>

<br>

---

## 👶 서비스 한 줄 소개
> **“복잡한 학사 정보를 대신 찾아주고 정리해주는 대학생 전용 AI 챗봇입니다.”**

양동이 챗봇은 동양미래대학교 학생들이  
**공지사항·학사일정·교내 정보**를 더 쉽고 빠르게 확인하도록 돕는  
**AI 기반 학사 정보 어시스턴트 서비스**입니다.

<br>

---

# 🌟 Why 양동이? (이 서비스가 필요한 이유)

## 👀 이런 경험 있지 않나요?

- 공지사항이 너무 많아서 **중요한 내용만 보고 싶을 때**
- “**시험 일정 언제지?**” / “**이번 주 강의실 어디야?**” 같은 질문이 계속 생길 때
- 학사 사이트가 여기저기 흩어져 있어 **원하는 정보를 찾기 어려울 때**
- 반복되는 질문인데, **매번 검색하거나 담당자에게 문의해야 할 때**
- 모바일로는 학사 정보를 보기가 **너무 불편할 때**

<br>

---

# 💡 양동이는 이런 문제를 해결합니다

### ✔ 공지사항 자동 요약
- 복잡한 공지사항을 **핵심만 추려서 요약**
- 중요한 내용만 빠르게 확인 가능

### ✔ 실시간 학사 Q&A
- “다음 주 일정 뭐 있어?”, “휴강했어?” 같은 자연어 질문 처리
- 학사 관련 자유 질의응답 지원

### ✔ 학사 일정·공지 자동 연동
- 시험 일정, 강의 시간표, 휴강/보강 정보 자동 수집
- 크롤링 기반 최신성 유지

### ✔ 즐겨찾기 & 대화 저장
- 자주 묻는 질문을 즐겨찾기로 관리
- 이전 대화 모두 저장/삭제 가능

### ✔ 모바일 UI 지원
- 스마트폰에서도 빠르게 확인 가능한 반응형 웹

# 🛠️ 주요 기능 요약

| 기능 | 설명 |
|------|------|
| 🧠 **자연어 질의응답 (RAG 기반)** | 학생 질문 → 공지사항·학사정보 검색 → OpenAI 모델이 정확하게 답변 |
| 📢 **공지사항 요약 기능** | 학교 공지 자동 분석 → 핵심만 요약하여 제공 |
| 📅 **학사 일정 연동** | 개강/휴강, 시험, 강의실, 교내일정 자동 수집 |
| ⭐ **즐겨찾기 기능** | 자주 묻는 질문을 저장해 빠르게 접근 |
| 💾 **대화 기록 관리** | 모든 대화를 저장/삭제 가능 |
| 🔊 **음성(STT/TTS)** | 음성으로 질문하고 음성으로 답변 듣기 |
| 📱 **반응형 UI 제공** | 모바일 & 태블릿에서도 최적화된 인터페이스 제공 |
